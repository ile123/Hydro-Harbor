### HYDRO HARBOR

### How to run

#### Local:
This is how to run the app on someone's local machine
- **Client:**
  1. `npm install`
  2. `npm run dev`
- **Server:**
  1. `npm install`
  2. `npm run dev`
- `sudo docker compose --profile local up`

#### Dockerized:
This is how to run the app with Docker
- `sudo docker compose --profile dockerized up`
